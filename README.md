# handtalk-challenge

## Topic: Language Detection

### To run the script, be sure you have installed both package:

```shell
pip3 install textblob
```

```shell
pip3 install pycountry
```



Then you can run with

```shell
python3 main.py
```

and follow the given Instruction.



For example, type:

- Eu gosto de café.
- I love coffee.
- Ich mag Kaffee.
- Me gusta el café.
